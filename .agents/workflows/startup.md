---
description: How to start the CampusVoice system securely
---

Follow these steps to ensure all security protocols and real-time tracking are active:

1. Stop all existing terminal processes.
2. Open exactly one terminal in the project root.
3. Run:
```bash
npm run dev
```
4. Access the system at: `http://localhost:5173/`
5. Login with your credentials. 
   - Admin: `admin@ggu.bus` / `ggubusadmin123`
   - Student: Your PIN or email.
   - Parent: Your personal email.
